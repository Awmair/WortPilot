# WortPilot Brand And Assets

## Name

Chosen product name: **WortPilot**

Meaning:

- `Wort` = word.
- `Pilot` = guided navigation through business German.

Positioning:

- Practical.
- Professional.
- Personal study cockpit.
- Built for client work, not tourist phrases.

## Icon

Source asset:

- `public/app-icon.png`

Derived PWA assets:

- `public/icon-192.png`

Prompt summary:

```text
Premium iOS-style app icon for a German business-language learning app for AI freelancers,
combining a lesson card, audio wave, and professional learning cues. Warm paper, deep ink,
emerald pronunciation accent, restrained gold and formal blue. No text, no flag, no watermark.
```

## Brand Voice

Short, direct, useful.

Examples:

- `Continue lesson`
- `Clear due reviews`
- `Hear it, say it, replay it`
- `Sync now`
- `Weak-point radar`

Avoid:

- Motivational fluff.
- Long onboarding copy.
- Tourist language framing.
- Generic gamification language.

## Asset Rules

- Do not overwrite generated source assets without keeping a versioned replacement.
- Project-bound image assets must live in the repo, not only under `.codex/generated_images`.
- Prefer code-native UI for controls and diagrams.
- Use raster generation only for icons, hero art, marketing preview images, or complex visual references.

## Future Asset Needs

- App Store-style screenshot set for install instructions.
- Dark-mode icon variant if Deep Focus becomes a main mode.
- Small empty-state illustrations for review complete, sync success, and offline mode.
